# Malakias Escasinas Interactive Portfolio

A ready-to-run Vite + React + TypeScript portfolio website using Malakias T. Escasinas' profile, resume details, and technical summary.

## Run locally

```bash
npm install
npm run dev
```

Open the URL printed by Vite, usually `http://localhost:5173/`.

## Build for production

```bash
npm run build
npm run preview
```

## Customize

Edit the profile content in `src/profileData.ts` and the interface in `src/App.tsx` / `src/App.css`.
